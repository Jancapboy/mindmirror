# PRD — 心镜 (MindMirror) · AI 心理健康助手

> **版本**: v0.1.0-MVP  
> **定位**: 心理学理论 × AI 对话的情绪健康陪伴平台  
> **核心差异**: 以循证心理学知识库为根基，AI 提供结构化、可追踪的心理支持，不是"陪你聊天"而是"帮你理解自己"

---

## 1. 产品概述

### 1.1 一句话描述
一个基于循证心理学知识库、AI 增强的个人心理健康管理工具，让心理自助从"网上搜鸡汤"变成"理论+测评+追踪+AI指导"的科学化陪伴。

### 1.2 目标用户
- 日常情绪波动者（压力大、焦虑、低落）
- 心理自助学习者（想系统了解心理学）
- 心理咨询来访者（辅助记录和追踪）

### 1.3 核心价值主张
> **"不是 AI 在安慰你，是 AI 在帮你用心理学工具理解自己"**

## 2. 功能模块

### 2.1 情绪日记（核心）

#### 2.1.1 每日记录
- **情绪评分**: 1-10 量表 + 情绪标签（开心、焦虑、悲伤、愤怒、平静...）
- **情境记录**: 发生了什么、在哪、和谁
- **身体感受**: 头痛、胸闷、失眠等躯体化症状
- **想法记录**: 自动化思维捕捉（CBT 核心）
- **能量水平**: 精力 1-10 评分

#### 2.1.2 趋势可视化
- **情绪曲线**: 近 7/30/90 天情绪波动图
- **热力图**: 情绪 × 时间/情境的交叉分析
- **周期识别**: 发现情绪规律（如每周一焦虑、经期前低落）
- **触发因子**: 统计哪些情境最常引发负面情绪

### 2.2 心理测评系统

#### 2.2.1 标准化量表
- **PHQ-9**: 抑郁筛查
- **GAD-7**: 焦虑筛查
- **PSS**: 知觉压力量表
- **WHO-5**: 幸福感指数
- **自尊量表**: Rosenberg SES
- **依恋类型测试**: 成人依恋量表

#### 2.2.2 测评流程
- 引导式答题 → 自动计分 → 结果解读 → 建议方案
- 历史对比：与上次测评对比变化
- 风险预警：高分自动提示关注

### 2.3 知识库系统（核心引擎）

> **所有 AI 回应都基于循证心理学知识，不是鸡汤生成器。**

#### 2.3.1 知识图谱结构

```
知识层（四层架构）
│
├─ 理论层
│  ├─ 认知行为疗法 (CBT)
│  ├─ 正念减压 (MBSR)
│  ├─ 接纳承诺疗法 (ACT)
│  ├─ 辩证行为疗法 (DBT)
│  ├─ 精神分析/心理动力学
│  ├─ 人本主义
│  └─ 积极心理学
│
├─ 技术层
│  ├─ 认知重构：自动化思维识别、证据检验
│  ├─ 暴露疗法：层级暴露、系统脱敏
│  ├─ 放松技术：腹式呼吸、渐进式肌肉放松
│  ├─ 正念练习：身体扫描、观呼吸、行走冥想
│  ├─ 情绪调节：STOP技术、DBT TIPP
│  ├─ 行为激活：活动计划、愉悦事件安排
│  └─ 社交技能： assertiveness 训练
│
├─ 测评层
│  ├─ 量表原始文本（授权/公开版本）
│  ├─ 计分规则
│  ├─ 常模数据
│  └─ 临床意义解读
│
└─ 危机干预层
   ├─ 自伤/自杀风险信号识别
   ├─ 危机应对步骤
   ├─ 求助资源（热线、医院）
   └─ 安全计划模板
```

#### 2.3.2 个性化推荐引擎
- 基于用户情绪数据 + 测评结果 → 推荐最适合的心理技术
- 例如：焦虑高分 → 推荐腹式呼吸 + 渐进式肌肉放松 + 暴露层级表

### 2.4 AI 对话疗愈

#### 2.4.1 对话模式
- **情绪急救**: 当下情绪困扰 → 即时技术引导
- **认知重构**: 识别自动化思维 → 证据检验 → 替代思维
- **每日检视**: 回顾一天 → 积极事件挖掘 → 感恩练习
- **技术教练**: 教授具体心理技术 → 现场练习 → 反馈调整
- **知识问答**: 心理学概念解释

#### 2.4.2 安全边界
- **风险识别**: 检测到自伤/自杀意念时 → 触发危机干预流程
- **免责声明**: 明确说明不替代专业心理咨询
- **转介机制**: 推荐本地专业资源

### 2.5 练习工具箱

- **呼吸练习**: 视觉化引导（圆环缩放模拟呼吸节奏）
- **身体扫描**: 语音引导 + 身体部位高亮
- **思维记录表**: CBT 经典五栏表（情境-情绪-自动化思维-证据-替代思维）
- **感恩日记**: 每日三件好事
- **情绪温度计**: 实时情绪评估小部件

### 2.6 进度追踪

- **练习 streak**: 连续练习天数
- **技能掌握**: 已学习/已练习的技术清单
- **目标设定**: SMART 目标 + 进度追踪
- **里程碑**: 达成成就（如"连续记录 7 天"、"完成首个量表"）

### 3.2 知识库实现

```typescript
// 心理学知识条目
interface PsychologyEntry {
  id: string;
  category: 'theory' | 'technique' | 'assessment' | 'crisis';
  title: string;
  content: string;
  evidenceLevel: 'strong' | 'moderate' | 'emerging' | 'anecdotal'; // 循证等级
  indications: string[];     // 适用症状（如焦虑、抑郁、失眠）
  contraindications: string[]; // 不适用情况
  steps?: string[];          // 操作步骤（技术类）
  resources?: string[];      // 推荐资源
  tags: string[];
}

// 用户画像（用于个性化推荐）
interface UserPsychProfile {
  primaryConcerns: string[];     // 主要困扰
  assessmentResults: Map<string, number>; // 量表历史结果
  practicedTechniques: string[]; // 已练习技术
  preferredApproach?: string;    // 偏好疗法流派
  riskFlags: string[];           // 风险标记
}
```

### 3.3 AI 对话安全架构

```
用户输入
    ↓
[安全过滤器] 检测危机信号（自伤/自杀关键词）
    ↓
  ├─ 高风险 → 危机干预流程（安抚 + 资源提供 + 建议求助）
  └─ 正常 → 继续处理
    ↓
[意图识别] 分类：情绪急救/认知重构/技术学习/知识问答
    ↓
[知识检索] 根据意图 + 用户画像 → 检索相关心理学知识
    ↓
[响应生成] AI 基于知识生成回应
    ↓
[输出检查] 确保不含医疗建议、含免责声明
```

## 4. UI/UX 设计

### 4.1 视觉风格
- **主题**: 疗愈系 × 现代简约
- **色彩**:
  - 主色：雾蓝 `#7ba7bc`（平静）、薄荷 `#98d8c8`（疗愈）
  - 辅色：暖白 `#faf9f6`、浅灰 `#e8e6e1`
  - 情绪色：焦虑-琥珀、抑郁-灰蓝、平静-薄荷、开心-暖黄
- **字体**: 圆润无衬线（亲和感）
- **动效**: 舒缓过渡、呼吸感动画（避免刺激）

### 4.2 核心页面

#### 4.2.1 今日首页
- 情绪快速记录入口（大按钮）
- 今日推荐练习（基于最近情绪数据）
- 情绪趋势迷你图
- 连续记录天数

#### 4.2.2 情绪日记页
- 日历视图（日期色块 = 当日主导情绪）
- 列表视图（可筛选、搜索）
- 单条记录详情（可编辑、可删除）

#### 4.2.3 测评中心
- 量表卡片列表
- 测评历史
- 结果对比图表

#### 4.2.4 AI 对话页
- 温暖的开场白
- 对话气泡（区分用户/AI）
- 快捷操作按钮（"我很难受"/"教我一个放松技巧"/"分析今天的情绪"）
- 练习嵌入：AI 可在对话中直接启动呼吸练习等交互组件

#### 4.2.5 练习工具箱
- 技术卡片网格
- 分类筛选（放松/认知/正念/行为）
- 练习记录

### 4.3 交互细节
- **情绪记录**: 滑动选择情绪强度，动画反馈
- **呼吸练习**: 圆环随呼吸缩放，配合舒缓背景音
- **危机干预**: 柔和但明确的提示，一键拨打热线

## 7. 非功能性需求

- **隐私**: 所有数据本地加密存储，密钥由用户掌握
- **离线可用**: 知识库和练习工具完全离线
- **AI 调用**: 仅在对话时联网，情绪记录本地处理
- **可及性**: 支持高对比度模式、大字体
- **响应速度**: 页面切换 < 200ms，AI 回应 < 5s

---

## 8. 伦理与安全

### 8.1 明确边界
- 产品定位：**心理健康自助工具**，非医疗诊断
- 所有量表结果标注："仅供参考，不构成诊断"
- AI 回应标注："基于心理学知识，不替代专业咨询"

### 8.2 危机干预
- 检测到高风险内容时：
  1. 立即显示安抚信息
  2. 提供 24 小时危机热线
  3. 建议联系信任的人或专业人士
  4. 提供安全计划模板

### 8.3 数据保护
- 本地优先：所有敏感数据不上传
- 加密：IndexedDB 数据 AES 加密
- 导出：用户可导出/删除自己的全部数据
